## About BilMod

BilMod is a modern web application aimed at simplifying online education. It provides tools for course management, interactive learning, and student progress tracking. The platform is built with flexibility and extensibility in mind, making it suitable for schools, universities, and independent educators.

---

## Features

- **User Authentication**: Secure registration and login for students and teachers.
- **Course Management**: Create, edit, and organize courses and modules.
- **Lesson Delivery**: Upload and manage multimedia lessons and resources.
- **Assignments & Quizzes**: Create assignments, quizzes, and automated grading.
- **Progress Tracking**: Monitor student progress and performance analytics.
- **Discussion Forums**: Facilitate communication between students and instructors.
- **Responsive Design**: Fully functional on desktop and mobile devices.
- **Admin Dashboard**: Manage users, content, and site settings.

---

## Tech Stack

- **Backend**: [Symfony](https://symfony.com/) (PHP)
- **Frontend**: Twig, HTML5, CSS3, JavaScript
- **Database**: MySQL / MariaDB
- **Other Tools**: Composer, Doctrine ORM, Webpack Encore

---

## Installation

1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/bilmod.git
    cd bilmod
    ```

2. **Install dependencies:**
    ```bash
    composer install
    npm install
    ```

3. **Configure environment:**
    - Copy `.env.example` to `.env` and update database credentials.

4. **Run database migrations:**
    ```bash
    php bin/console doctrine:migrations:migrate
    ```

5. **Start the development server:**
    ```bash
    symfony server:start
    ```

## Usage

- Access the platform at `http://localhost:8000`
- Register as a new user or log in with existing credentials.
- Explore courses, enroll, and start learning!

### À propos de BilMod

BilMod est une application web moderne visant à simplifier l'éducation en ligne. Elle propose des outils pour la gestion des cours, l'apprentissage interactif et le suivi des progrès des étudiants. La plateforme est conçue pour être flexible et extensible, adaptée aux écoles, universités et formateurs indépendants.

---

### Fonctionnalités

- **Authentification des utilisateurs** : Inscription et connexion sécurisées pour étudiants et enseignants.
- **Gestion des cours** : Création, modification et organisation des cours et modules.
- **Diffusion des leçons** : Gestion de contenus multimédias et ressources pédagogiques.
- **Devoirs & Quiz** : Création de devoirs, quiz et correction automatisée.
- **Suivi des progrès** : Analyse des performances et suivi des étudiants.
- **Forums de discussion** : Communication facilitée entre étudiants et enseignants.
- **Design responsive** : Fonctionnalités complètes sur ordinateurs et mobiles.
- **Tableau de bord admin** : Gestion des utilisateurs, contenus et paramètres du site.
